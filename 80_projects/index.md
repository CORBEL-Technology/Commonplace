# Active projects

The live tracker for this workspace's active projects: one folder per project, each with an
`index.md` (status + links) and a `loops.md` (its outstanding loops). The SessionStart brief and the
daily brief aggregate the open loops from here. This is live working state (C1), distinct from
durable knowledge in `15_canon/` and from any gitignored root `projects/` (self-contained
apps/binaries).

Seed a project only when it has live content (`AGENTS.md` rule 8); add a row below when you do.
Cross-cutting loops not tied to one project live in `_general/loops.md`.

* [How the folder works](README.md) - what a project folder is, how loops open and close, the boundary vs canon
* [Project & loops schema](../30_schemas/project.md) - the folder contract (index.md + loops.md, the loop table shape)
* [Cross-cutting loops](_general/loops.md) - loops tied to no single project
