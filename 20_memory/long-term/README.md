# Long-term memory (depth 3)

Durable, canonical knowledge: verified facts, founder preferences, procedures/SOPs, tool-recipes,
stable relationships, landmark decisions, and recurring patterns. **One canonical card per
fact-family** (gist over verbatim) — repeated evidence strengthens the one card, it does not breed
duplicates. Each card stays source-linked back to the journal.

This layer is **selective by design** — it gets sharper, not larger. A card earns its place by being
corroborated, reused, and stable, or by an explicit `pivotal` mark. **`pivotal` / `do_not_drop`
cards bypass decay entirely** — the structural guarantee that the pivotal points from years ago are
kept while the surrounding noise decays out of short-term. Updates create supersession chains, never
overwrites. `class:` (observational/procedural/normative) is a frontmatter tag here.

Starts empty. The first durable preferences and SOPs are minted here as the workspace runs (e.g. a
`founder-preferences` card once preferences are stated and captured back).
