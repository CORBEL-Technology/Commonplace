If this instance is replacing or absorbing an older workspace, record the migration here as a
**curated fresh start, not a port**: the old workspace is an archive to cherry-pick from, and the
burden of proof is on **inclusion**, not omission. Recommended discipline:

- **Method per section:** a read-only scout produces a decision-ready triage table; the owner
  decides each row (**pull** / **archive-ref** / **leave**). Pulled items get a provenance journal
  entry; deliberate omissions get a one-line entry under "Explicitly left behind" so they are
  decisions, not accidents.
- **Never break the evidence chain:** an atom imported in a phase must point at a real journal entry.
- **Re-verify live state after each phase:** trust no status doc, including this one.
- **Keep the old workspace authoritative** until each phase is confirmed; retire it only when the
  migration completes.
